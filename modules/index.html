terra.api

The api module is wrapping some endpoints from Terra lcd server.

You can change the api endpoint used by setting:

terra.api.Client.URL = "https://my.lcd.server"

Client:

Client()

Client.get(
    path: str,
    params: Optional[Dict[str, str]] = None,
    timeout: Optional[int] = 5,
) -> dict

Client.post(
    path: str,
    json: Optional[Dict[str, str]],
    params: Optional[Dict[str, str]] = None,
    timeout: Optional[int] = 10,
) -> dict

Note: You should probably not have to use this class. It is meant for internal use.
